# CheckBill

เว็บหารค่าอาหาร — โคลนจาก [jabont.com/checkbill](https://jabont.com/checkbill/)

## ฟีเจอร์

- เพิ่มรายการอาหาร + ราคา (พร้อมเครื่องคิดเลข)
- เลือกคนที่กินแต่ละรายการ (หารไม่เท่ากันได้)
- ปัดเศษทศนิยมขึ้นอัตโนมัติ
- ติดตามยอดแต่ละคน + ทำเครื่องหมายจ่ายแล้ว
- PromptPay QR (ผ่าน promptpay.io)
- แชร์บิลผ่าน URL / บันทึกใน localStorage

ยังไม่เชื่อม Supabase — ข้อมูลเก็บในเบราว์เซอร์เท่านั้น

## เริ่มใช้งาน

```bash
npm install
npm run dev
```
